# Connect 4 - vs Magic AI

A beautiful Connect 4 game with an unbeatable AI opponent powered by Minimax algorithm.

## Quick Start

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start development server:**
   ```bash
   npm start
   ```

3. **Open browser:** http://localhost:3000

## Deploy to Vercel

1. **Push to GitHub**
2. **Go to vercel.com**
3. **Import from GitHub**
4. **Deploy!**

## Features

✨ Unbeatable Magic AI (Depth 7)
🔊 Sound Effects
📊 Stats Tracking
🎨 Beautiful UI
📱 Responsive Design
